Breadth-First Search

Explores all neighbors at current depth before moving deeper. Uses a queue. Time: O(V+E), Space: O(V).

O(V + E) Queue
🔍

Depth-First Search

Explores as far as possible along each branch before backtracking. Uses recursion/stack. Time: O(V+E), Space: O(V).

O(V + E) Stack
📍

Dijkstra's Algorithm

Finds shortest path from source to all vertices in weighted graph. Greedy approach with priority queue. Time: O((V+E) log V).

O((V+E) log V) Greedy
🌳

Minimum Spanning Tree

Prim's and Kruskal's algorithms find MST connecting all vertices with minimum total edge weight. Critical for network design.

O(E log V) Union-Find
