Activity Selection

Select maximum number of non-overlapping activities. Sort by finish time, greedily pick earliest-finishing compatible activity.

O(n log n) Sort First Optimal
🎒

Fractional Knapsack

Take fractions of items to maximize value. Sort by value/weight ratio, fill knapsack greedily. Unlike 0/1 knapsack, this has greedy optimal.

O(n log n) Value/Weight Fractions OK
🌳

Huffman Coding

Build optimal prefix-free binary code. Merge lowest frequency nodes greedily. Foundation of ZIP, JPEG, MP3 compression.

O(n log n) Min-Heap Prefix-Free
💼

Job Sequencing

Maximize profit by scheduling jobs before deadlines. Sort by profit, place each job at latest available slot before deadline.

O(n²) Deadline Max Profit
🪙

Coin Change (Greedy)

Always pick largest denomination first. Works for canonical coin systems (US, EU). Not always optimal for arbitrary denominations.

O(n) Canonical Not Always Opt
🛣️

Dijkstra's Algorithm

Greedy selection of minimum distance vertex. Works because choosing local minimum preserves global optimality for non-negative weights.

O((V+E) log V) Priority Queue Non-Negative
