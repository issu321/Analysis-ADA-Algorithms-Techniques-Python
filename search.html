Searching Algorithms

Visualize how different search strategies probe the array. Purple = Current, Yellow = Mid, Green = Found, Red = Not Found.

Current Mid Found Not Found
⚡

Binary Search

Halves search space each iteration. Requires sorted array. The gold standard for searching sorted data. Iterative and recursive variants.

O(log n) O(1) Space Sorted Only
📏

Linear Search

Checks each element sequentially. Simple, works on unsorted data. Best for small arrays or when data is unordered.

O(n) O(1) Space Unsorted OK
🦘

Jump Search

Jumps ahead by √n steps, then linear searches the block. Between linear and binary in performance. Good for sorted arrays.

O(√n) O(1) Space Block Strategy
🎯

Interpolation Search

Estimates position using value distribution. Best for uniformly distributed sorted data. Can achieve O(log log n) in ideal cases.

O(log log n) Uniform Data Estimate Pos
📈

Exponential Search

Finds range by repeated doubling (1, 2, 4, 8...), then binary searches within it. Efficient for unbounded/infinite arrays.

O(log n) Unbounded Range Finding
🔢

Ternary Search

Divides array into three parts. Useful for finding maximum of unimodal functions. Slightly slower than binary in practice.

O(log₃ n) Unimodal 3-Way Split
