ADA Theory & Analysis

Asymptotic notation, Master Theorem, NP-Completeness, and algorithm design paradigms. The mathematical foundation of efficient computation.

📐 Asymptotic Notation

Notation Name Definition Example
O(f(n)) Big-O (Upper Bound) T(n) ≤ c·f(n) for n ≥ n₀ 3n² + 5n = O(n²)
Ω(f(n)) Big-Omega (Lower Bound) T(n) ≥ c·f(n) for n ≥ n₀ n³ + 2n = Ω(n³)
Θ(f(n)) Big-Theta (Tight Bound) c₁·f(n) ≤ T(n) ≤ c₂·f(n) 2n² + 3n = Θ(n²)
o(f(n)) Little-o (Strict Upper) lim T(n)/f(n) = 0 n = o(n²)
ω(f(n)) Little-omega (Strict Lower) lim T(n)/f(n) = ∞ n² = ω(n)

🎓 Master Theorem

For recurrences of the form T(n) = a·T(n/b) + f(n) where a ≥ 1, b > 1:

Case 1

f(n) = O(n^(log_b a - ε))

T(n) = Θ(n^(log_b a))

Leaf work dominates. Example: Merge Sort T(n)=2T(n/2)+n → Θ(n log n)

Case 2

f(n) = Θ(n^(log_b a))

T(n) = Θ(n^(log_b a) · log n)

Work balanced across levels. Example: T(n)=2T(n/2)+Θ(n)

Case 3

f(n) = Ω(n^(log_b a + ε))

T(n) = Θ(f(n))

Root work dominates. Example: T(n)=2T(n/2)+n² → Θ(n²)

🔬 Complexity Classes

P

Polynomial time. Solvable in O(n^k) for some constant k. The "easy" problems.

Sorting, Shortest Path, MST, GCD

NP

Nondeterministic Polynomial. Solutions verifiable in polynomial time. Contains P.

Sudoku, Graph Coloring, Subset Sum

NP-Complete

Hardest problems in NP. Every NP problem reduces to these in polynomial time.

SAT, 3-SAT, Clique, Vertex Cover, TSP

NP-Hard

At least as hard as NP-Complete. May not be in NP. Includes optimization variants.

Halting Problem, Chess, TSP Optimization

🧠 Design Paradigms

🔀

Divide and Conquer

Break problem into smaller subproblems, solve recursively, combine results. Merge Sort, Quick Sort, Binary Search, Strassen's Matrix Multiplication.

T(n) = aT(n/b) + f(n) Master Theorem
🧩

Dynamic Programming

Optimal substructure + overlapping subproblems. Memoize or tabulate to avoid recomputation. Knapsack, LCS, Edit Distance, Matrix Chain.

Memoization Tabulation Optimal
💎

Greedy Method

Make locally optimal choice at each step. Works when greedy choice property holds. Activity Selection, Huffman, Fractional Knapsack, Dijkstra.

Local Optimal Fast Not Always Opt
🔙

Backtracking

Build solution incrementally, abandon partial candidates ("backtrack") when they cannot possibly lead to valid solution. N-Queens, Sudoku, Graph Coloring.

State Space Pruning Exponential
🌿

Branch and Bound

Systematically enumerate candidate solutions using bounds to prune branches. Optimization version of backtracking. TSP, Job Assignment.

Bounds Pruning Optimization
🧬

Randomized Algorithms

Use randomness as part of the logic. Las Vegas (always correct, random time) vs Monte Carlo (fixed time, probabilistic correctness). QuickSort pivot, Miller-Rabin.

Las Vegas Monte Carlo Expected Time
