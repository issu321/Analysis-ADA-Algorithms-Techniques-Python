Bubble Sort

Repeatedly steps through list, compares adjacent elements and swaps them if wrong order. Simple but inefficient for large data.

O(n²) Stable In-Place
⚡

Quick Sort

Divide-and-conquer with pivot selection. Partitions array around pivot, recursively sorts subarrays. Fastest general-purpose sort.

O(n log n) Unstable In-Place
🔄

Merge Sort

Divides array in half, recursively sorts, then merges. Guaranteed O(n log n). Stable but requires O(n) extra space.

O(n log n) Stable O(n) Space
🏗️

Heap Sort

Builds max-heap, repeatedly extracts maximum and heapifies. In-place with guaranteed O(n log n). Not stable.

O(n log n) Unstable In-Place
🎯

Counting Sort

Non-comparison sort using frequency counting. O(n+k) time where k is range. Stable and efficient for small integer ranges.

O(n+k) Stable Integer Only
🔢

Radix Sort

Processes digits from least significant to most. Uses counting sort as subroutine. O(d(n+k)) for d digits. Stable.

O(d(n+k)) Stable Non-Comparison
